# Increment Decrement Counter

A simple counter application built with HTML, CSS, and JavaScript.

## Features

- Increase the counter value
- Decrease the counter value
- Reset the counter to zero
- Responsive and easy-to-use interface

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Use

1. Open the application in your browser.
2. Click the **Increase** button to increment the counter.
3. Click the **Decrease** button to decrement the counter.
4. Click the **Reset** button to set the counter back to zero.

## Project Purpose

This project was created to practice JavaScript DOM manipulation and event handling.
